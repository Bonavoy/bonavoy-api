# Bonavoy GraphQL API

Repo for Bonavoy's GraphQL API

# Setup

1. clone repo and `cd` into root
2. get secret files
3. install node modules with `yarn install`
4. run with `yarn dev`

# Testing

Run `yarn test` or `yarn test:watch`

# Documentation

- Local: http://localhost:4000/graphql
- Prod: http://api.bonavoy.com/graphql
