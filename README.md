# Bonavoy REST API

Repo for Bonavoy's REST API

# Setup

1. clone repo and `cd` into root
2. get secret files
3. install node modules with `yarn install`
4. run with `yarn dev`

# Testing

Run `yarn test` or `yarn test:watch`

# Documentation

- Local: http://localhost:4000/docs
- Prod: http://api.bonavoy.com/docs
